# Uniswap API & SDK Testing Project

A Vue 3 + TypeScript project for testing and experimenting with Uniswap's GraphQL API and SDK. This project provides a swap interface UI that integrates with Uniswap's decentralized exchange infrastructure.

## Overview

This project serves as a testing ground for:

- **Uniswap GraphQL API**: Testing queries for token data, prices, swap quotes, and pool information
- **Uniswap SDK**: Integration with Uniswap's SDK for token swaps and liquidity operations
- **Swap Interface**: A modern, dark-themed UI for token swapping (QUBIC/USDT pairs)
